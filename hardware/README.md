# Hardware

This directory contains the hardware design files for the early hardware wallet learning project.

The design is provided for educational and research purposes only.  
It is not intended to be used as a production wallet or to store real assets.

## Structure

```text
kicad/
  KiCad project files, including schematic and PCB layout.

bom/
  Bill of materials and component list.

gerber/
  Gerber files for PCB manufacturing.

schematic/
  Exported schematic files, such as PDF or image versions.

pcb/
  Exported PCB layout images or manufacturing notes.
